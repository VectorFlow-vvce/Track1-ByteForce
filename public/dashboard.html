Voice Emergency Monitor

Voice Detection Offline Capable

Uses Web Speech API to detect emergency keywords in real-time. Works offline after initial setup. Monitored keywords: help, emergency, save me, doctor, fall, pain, hurt

Listening for keywords...
How It Works

1. Microphone listens continuously on device

2. Speech-to-text runs locally (offline)

3. Keywords matched against emergency list

4. Local alert triggered instantly

5. Syncs to server when internet available

6. Caregiver dashboard updates in real-time

🔒 Privacy Note
Audio is processed locally on the device. No recordings are stored or transmitted. Only detected keywords trigger alerts.
